# Выход пользователя

### URL: `POST` `/logout`

### Параметры

> Запрос на выход пользователя из системы не требует параметров.

### Ответы

>| Код HTTP | Тип контента       | Ответ                       |
> |----------|--------------------|-----------------------------|
> | `200`    | `application/json` | `{"message": "Logged out"}` |
